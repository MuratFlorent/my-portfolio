# Pack projets Light Developer

Ce pack contient :
- un catalogue JSON des projets
- un stub Markdown par projet prêt à intégrer dans Astro Content
- un template pour les futures missions plus petites
- un brief de tasks pour Junie

## Idée d'intégration Astro
- `src/assets/projects/...` : images
- `src/content/projects/...` : stubs markdown
- `src/content/config.ts` : schema collection
- `src/pages/projets/[slug].astro` : page dédiée
- `src/pages/projets/index.astro` : listing complet
